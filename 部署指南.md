# 智选未来 — 部署指南

## 临时外网访问（立即可用）

**访问地址**: https://full-tools-create.loca.lt

> 注意：这是临时隧道，服务停止后链接失效。如需长期访问，请使用下方永久部署方案。

---

## 方案一：Cloudflare Pages（推荐，免费永久）

### 步骤：
1. 访问 https://dash.cloudflare.com 注册/登录
2. 左侧菜单点击 **Pages** → **创建项目**
3. 选择 **直接上传**
4. 上传 `zhixuan-weilai-deploy.zip` 文件
5. 项目名填写 `zhixuan-weilai`（或自定义）
6. 点击 **部署站点**
7. 获得 `https://zhixuan-weilai.pages.dev` 域名

### 绑定自定义域名（可选）：
- 在 Pages 项目设置中添加自定义域名
- 按提示修改 DNS 记录
- 自动获得 HTTPS 证书

**优点**：全球 CDN、国内访问快、免费、自动 HTTPS

---

## 方案二：GitHub Pages（免费永久）

### 步骤：
1. 访问 https://github.com 创建新仓库
2. 仓库名填写 `zhixuan-weilai`
3. 解压 `zhixuan-weilai-deploy.zip`，将文件上传到仓库
4. 仓库 Settings → Pages
5. Source 选择 **Deploy from a branch**
6. Branch 选择 `main` / `root`
7. 保存后访问 `https://你的用户名.github.io/zhixuan-weilai`

**优点**：免费、与 Git 集成
**缺点**：国内访问偶尔不稳定

---

## 方案三：Vercel（免费永久）

### 步骤：
1. 访问 https://vercel.com 用 GitHub 账号登录
2. 点击 **Add New Project**
3. 导入 GitHub 仓库（或拖拽上传文件夹）
4. Framework Preset 选择 **Other**
5. 点击 **Deploy**
6. 获得 `https://zhixuan-weilai.vercel.app` 域名

**优点**：部署极快、自动 HTTPS、全球 CDN

---

## 方案四：国内服务器（Nginx）

如果你有阿里云/腾讯云服务器：

```bash
# 1. 上传文件到服务器
scp zhixuan-weilai-deploy.zip root@你的服务器IP:/var/www/

# 2. 服务器上解压
ssh root@你的服务器IP
cd /var/www
unzip zhixuan-weilai-deploy.zip

# 3. Nginx 配置 /etc/nginx/sites-available/zhixuan
cat > /etc/nginx/sites-available/zhixuan << 'EOF'
server {
    listen 80;
    server_name 你的域名.com;
    root /var/www/zhixuan-weilai-deploy;
    index index.html;
    location / {
        try_files $uri $uri/ /index.html;
    }
}
EOF

# 4. 启用配置
ln -s /etc/nginx/sites-available/zhixuan /etc/nginx/sites-enabled/
nginx -s reload
```

## 方案五：阿里云 OSS + CDN（推荐国内用户）

### 步骤：
1. 登录阿里云控制台 → 对象存储 OSS
2. 创建存储桶，地域选择靠近用户的区域
3. 基础设置 → 静态页面 → 默认首页设置为 `index.html`
4. 上传解压后的文件
5. （可选）绑定 CDN 加速域名

**优点**：国内访问极快、成本极低

## 注意事项

1. **本项目是纯前端静态页面**，无需后端服务器
2. 所有数据保存在 JS 文件中，无需数据库
3. 推荐使用 **Cloudflare Pages** 或 **阿里云 OSS** 获得最佳国内访问体验
4. 如需更新内容，修改对应 JS 文件后重新部署即可
